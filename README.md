# Интерполяция функций полиномами
## Структура проетка:
1. check_nodes.f90  функция для проверки значения в узлах. Сравнивает с узлами равномерной сетки, поэтому для чебышевской не всегда проверяются вообще какие-либо узлы.
2. exact.f90 функция генерации абсолютных значений на равномерной сетке.
3. generate.f90 функция генерации тестовых данных
4. interpolation_mod.f90 основная функция интерполяции
5. precision_mod.f90 точность
6. test_functions_mod.f90 функции для тестирования (Рунге, всплеск, синус...)

## Запуск программы
Протестировать встроенные функции
```bash
make test
```

Нарисовать графики для встроеных функций
```bash
make plot
```

Только для интерполяции
```bash
make interpol
./interpol {uniform/chebyshev}
```
